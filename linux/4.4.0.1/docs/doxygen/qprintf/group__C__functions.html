qprintf_API void qprintf_set_mask ( unsigned long long  high,
unsigned long long  low 
)

Set the mask controlling which bytes to display when printing out an HVX register.

If the nth bit of mask is set, the nth byte of HVX will be displayed. When printing HVX as 16-bit or 32-bit elements, only the bit corresponding to the lowest byte of the element controls whether the element will be printed out or not.

Parameters
highMask for upper 64 bytes of HVX vector.
lowMask for lower 64 bytes of HVX vector.
Returns
None.

Example:

  • Display the 32-bit odd elements of the 64 most significant bytes and the even bytes of the 64 least significant bytes of HVX vectors printed with option %m.
    1 // From C before invoking your assembly routine
    2 qprintf_set_mask(QPRINTF_MASK_ODD_32,QPRINTF_MASK_EVEN_8);
    3 
    4 // From assembly
    5 qprintf("v0: %mx",v0);

See also HVX registers for more assembly examples using %m.
