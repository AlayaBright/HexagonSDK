int16_t qhmath_atan2_h ( int16_t  x,
int16_t  y 
)

4-quadrant arctan Returns the principal value of the arctan of y/x, expressed in radians. To compute the value, the function takes into account the sign of both arguments in order to determine the quadrant.

Parameters
[in]xValue representing the proportion of the x-coordinate in 16-bit fixed-point Q15 format.
[in]yValue representing the proportion of the y-coordinate in 16-bit fixed-point Q15 format.
Returns
Principal arctan of y/x, in the interval [-1.0,+1.0] radians. NOTE: Output result is y ~ atan(y/x)/pi
